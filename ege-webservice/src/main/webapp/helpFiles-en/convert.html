Help: Select file and convert

Please browse for the file you wish to convert. After you are done, please add any additional images, if required. You can set some more options after clicking on "Show advanced options". When you think everything is ok, please click "Convert". After you have done so, please wait while your document is being converted. When it is done, you will be notified and then you will be able to download the converted document. Please DO NOT close the window before the conversion is finished. If you wish to get back to the first screen, please press "Reset".
