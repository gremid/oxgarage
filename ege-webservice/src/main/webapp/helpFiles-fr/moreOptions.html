Help: Advanced options

Options avancées

Cette option vous permet (entre autres) de sélectionner un profile de conversion et de gérer les fichiers images associés avec un document. Elle permet également la visualisation des notices concernant chaque étape d'une séquence de transformations prédéfinie.

Profiles - You can choose different profiles in some conversions. Each profile produces a slightly different looking document(e.g. page headers, or some different styles)

Convert text only - Selectionnez cette option si vous souhaitez supprimer dans votre document converti toute référence à un image indiqué dans le document. (Cette option est déconseillée pour les formats bureautiques ODT, DOC, DOC, EPUB, etc.)

Download images - Selectionnez cette option si vous souhaitez inclure dans votre document converti les images indiqués par des références externes (par exemple sur l'Internet) dans le document. Cette option est selectionnée par défaut, et fortement conseillée pour les formats bureautiques ODT, DOC, DOC, EPUB, etc.

Copy and rename images - Selectionnez cette option si vous souhaitez inclure dans votre document converti les fiches images que vous avez chargé. Notez que les noms de ces fichiers images seront modifiés par le logiciel. Cette option est selectionnée par defaut, et fortement conseillée pour les formats bureautiques ODT, DOC, DOC, EPUB, etc.

URL for conversion with properties - This just shows what url will be the file sent to in order to perform the conversion. You do not have to worry about this, unless you would like to create your own webclient or perform the conversions with a commandline script.

URL for conversion with properties - Cette option affiche un URL correspondant à la transformation souhaitée. Cet URL peut servir à effectuer la transformation en partant d'un web service ou script.

URL for conversion without properties - Cette option affiche un URL correspondant à la transformation par défaut indiquée, sans prendre en compte les options selectionnées. Cet URL peut servir à effectuer la transformation en partant d'un web service ou script.
