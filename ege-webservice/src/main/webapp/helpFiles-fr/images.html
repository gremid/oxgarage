Chargement des images

Un document XML peut contenir des liens (références) sur des fichiers images qui sont stockés soit sur votre machine, soit ailleurs sur l'Internet. Dans le premier cas, il faut charger ces fiches images au même temps que le document lui même. Vous pouvez les charger un par un, ou en forme d'une archive zip qui les rassemble tous. Dans le deuxième cas, le logiciel de transformation va rechercher les fiches images au lieux indiques et vous n'aurez pas besoin de les charger. Nota: ces actions peuvent être modifiées (voir Options avancées)
